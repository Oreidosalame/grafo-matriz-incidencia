# Representação de Grafos - Matriz de Incidência

Este repositório contém um programa desenvolvido em **C** para a criação e manipulação de um grafo não direcionado, utilizando a estrutura de dados de **Matriz de Incidência**.

## ⚙️ Funcionalidades

O programa conta com um menu interativo via terminal que permite ao usuário:

* **Inserir aresta:** Conecta dois vértices no grafo.
* **Mostrar matriz:** Exibe visualmente a matriz de incidência no console.
* **Grau de um vértice:** Calcula e exibe a quantidade de arestas conectadas a um vértice específico.
* **Graus de todos os vértices:** Lista o grau de todos os vértices existentes no grafo.
* **Listar arestas:** Mostra todas as conexões criadas detalhadamente.
* **Limpar grafo:** Zera a matriz e reinicia a contagem de arestas.
